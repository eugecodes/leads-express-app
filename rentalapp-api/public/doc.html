• Endpoints Short List
Manatee API
  • System Users
    • This is a group of api endpoints that manage the administration of the system users.
    /register It creates a new user for the system. It isn't for leads only for system users. 
                                    POST http:///register HTTP/1.1
                                Content-Type: application/json
                                    
                                {
                                    "fullname": FULL_NAME,
                                    "user": USERNAME,
                                    "password":PASSWORD,
                                    "email": EMAIL,
                                    "phone": PHONE_NUMBER
                                }
    /login It's the user authentication, after validated the user credentials, it returns a token, that will be needed for every new operation with the api. 
                                    POST http:///login HTTP/1.1
                                Content-Type: application/json
                                
                                {
                                    "user": USERNAME,
                                    "password": PASSWORD
                                }
    /get-user Returns the user details, such as name, email, phone, maybe avatar later. 
                                    GET http:///get-user HTTP/1.1
                                Content-Type: application/json
                                Authorization: OAuth 2.0 JWT token
